# Contributing Guidelines - AIC-Recovery

Thank you for your interest in contributing to **AIC-Recovery** — the official Recovery & Resilience Layer of Adaptive Intelligence Circle & Human Meaning Network.

### Core Principles (Non-Negotiable)
- All contributions must preserve **third path absolute**, **zero-donation**, and **ethical-from-kernel**.
- Recovery mechanisms must prioritize ethical integrity and system sovereignty.
- The project remains non-profit and independent.

### How to Contribute
1. Fork the repository.
2. Create a feature branch for your changes.
3. Make your changes with clear documentation and tests.
4. Open a Pull Request (PR) with detailed explanation.

**During the Founder’s mandatory military service (2027–2029):**  
Only maintenance, bug fixes, documentation, and security updates are accepted. No major architectural changes without explicit Founder approval.

### What We Do NOT Accept
- Features that reduce resilience or introduce single points of failure.
- Contributions that weaken ethical recovery principles.
- Any form of commercial sponsorship or dual-licensing attempts.

By submitting a contribution, you agree to this Contributing.md, the Code of Conduct, and all policies in the `/POLICIES` folder.
