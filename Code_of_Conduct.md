# Code of Conduct - AIC-Recovery

We expect all contributors to act with respect, honesty, and integrity while working on the Recovery & Resilience Layer of Adaptive Intelligence Circle & Human Meaning Network.

### Core Principles (Non-Negotiable)
- Respect **third path absolute**, **zero-donation**, and **ethical-from-kernel**.
- Recovery mechanisms must never compromise user sovereignty or ethical integrity.
- Privacy and system resilience are fundamental rights, not optional features.

### Expected Behavior
- Be respectful and constructive in all interactions.
- Focus on improving resilience and ethical recovery.
- Clearly document any changes that affect recovery logic.
- Report security or ethical issues responsibly.

### Unacceptable Behavior
- Harassment, discrimination, or personal attacks.
- Proposing features that weaken ethical kernel or introduce centralization.
- Any attempt to bypass or weaken recovery safeguards.
- Commercial or dual-licensing proposals.

Violations may result in immediate removal from the project.

This Code of Conduct follows the Contributor Covenant with additional AIC core principles.
