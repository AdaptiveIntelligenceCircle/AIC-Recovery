# Examples - AIC-Recovery

This folder contains practical examples and scenarios demonstrating how **AIC-Recovery** works in real situations.

**Purpose**:  
Help contributors, maintainers, and users understand the recovery mechanisms, ethical constraints, and integration patterns of AIC-Recovery.

### Available Examples

- **node-failure-recovery.md** — How the system recovers when a node fails or goes offline
- **malicious-node-detection.md** — Detecting and isolating malicious or compromised nodes
- **ethical-rollback-example.py** — Safe and ethical rollback of compromised state
- **self-healing-demo.md** — Demonstration of autonomous self-healing
- **integration-with-hmn.md** — How Recovery integrates with Human Meaning Network

All examples strictly follow AIC’s core principles: ethical-from-kernel, third path absolute, and zero-donation.

**Note**: These are illustrative examples. Production use should always go through full testing and ethical validation.
